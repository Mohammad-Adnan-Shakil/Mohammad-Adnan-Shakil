## ◈ About Me

Second-year CSE student at Presidency University, Bengaluru.

I build backend-first systems where the architecture is a deliberate decision, not a default. My flagship project DeltaBox runs a Java Spring Boot API that serializes ML requests to JSON, calls a Python subprocess via ProcessBuilder, reads predictions from STDOUT, and returns structured responses — all within a single-server setup with JWT + RBAC security built from scratch.

## 🔴 DeltaBox — F1 Intelligence Platform

AI-powered F1 analytics platform. Java backend calls Python ML models via subprocess orchestration — not a microservice, not an API call. Direct cross-language execution with JSON over STDIN/STDOUT, keeping inference latency under 200ms with no external service dependency.

**Architecture decisions that matter:**
- ✅ ProcessBuilder subprocess execution — Java → Python JSON pipeline
- ✅ 3-model ensemble (XGBoost + Random Forest + Linear Regression) with disagreement flagged as high-uncertainty signal
- ✅ JWT + RBAC security built from scratch — no Spring Security defaults
- ✅ What-if simulation engine + response caching layer
- ✅ 87.3% prediction confidence across 24 races, 20 drivers
